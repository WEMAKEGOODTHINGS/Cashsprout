A mini Python-based banking system with a sleek GUI and MySQL backend for managing accounts, deposits, withdrawals, and transactions. Perfect as a beginner-friendly yet functional banking project.

Features
Create Accounts: Add new users with username, PIN, and initial deposit.
Secure Login: Authenticate using username and PIN.
Deposit & Withdraw: Simple money management with validations.
Check Balance: Instantly view account balance.
Transaction History: View all deposits and withdrawals for each account.
Interactive GUI: Modern, neon-inspired Python Tkinter interface.
